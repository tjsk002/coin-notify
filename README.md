### 기본 스택
| Language  | Java |
|-----------| --- |
| Framework | Spring Boot + WebFlux |
| 실시간 전송    | SSE or WebSocket |
| 빌드        | Gradle |

# 요구사항

> 프론트 구성할 필요 없이 백엔드만 개발
>

### **1. 사용자 관심 종목 등록**

- 관심 코인 설정
- 여러 종목 등록 가능

### **2. 실시간 시세 수집**

- 외부 API (예: 업비트)에서 일정 주기로 시세 수집
- 관심 종목 가격 변경 여부 확인

### **3. 실시간 알림 전송**

- SSE 또는 WebSocket으로 알림 푸시
- 사용자별 연결 관리 (아이디로 구분)
